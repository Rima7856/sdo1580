# SDO Клиент на основе взаимодействия с сайтом школы № 1580



## Авторизация и сессии
 Скрипт использует данные из .env для авторизации в системе SDO. 

## Умное управление сессиями:
 Скрипт сохраняет куки в файл cookies.pkl, чтобы не вводить логин/пароль при каждом запуске. Проверка валидности: автоматически определяет, не истекла ли текущая сессия, и обновляет её при необходимости.

## Взаимодействие с тестами:
 Поиск активных полей: скрипт сам находит sesskey и идентификатор нужного поля ввода (qX:X_answer). 

## Отправка решения: 
 Имитирует ручной ввод и нажатие кнопки «Проверить» (поддерживает sequencecheck для защиты от сбоев).Обработка CodeRunner: адаптирован для работы с заданиями типа CodeRunner, требующими мгновенной проверки кода.

## Обработка результатов:
 Программа извлекает результаты тестов (входные данные, ожидаемый и полученный ответы) прямо из ответа сервера (удаляя лишние HTML-теги и спецсимволы, оставляя в консоли только чистый текст). В выводе содержится только выводимые данные по вашему тесту, при необходимости можно удалить комментарии и высылать все тесты

```
### 2. Добавьте файл .env (измените .env.example под себя)

### 3. Активируйте виртуальное окружение

### 4. Скачайте необходимые библиотеки

```
